Gen Z Mental Wellness and Digital Lifestyle Patterns


Domain
Human - Technology Interaction

📌 Project Overview : 
  	This project analyzes the impact of digital lifestyle habits on the mental wellness of Gen Z individuals. The dataset was cleaned and analyzed using Google Colab (Python), and the processed data was imported into Power BI to build an interactive dashboard for visualizing trends, patterns, and key insights.

🔄 Workflow :
  	📥 Data Collection → 🧹 Data Cleaning → 📊 EDA → 📈 Visualization → 📋 Power BI Dashboard → 💡 Insights → 🎯 Recommendations → ✅ Final Outcome.

🛠️ Tools & Technologies :
Microsoft Power BI Microsoft Excel Data Visualization Data Analysis KPI Dashboard Design.

📊 Dashboard Features :
	📌 KPI Cards – Display key metrics at a glance
	📈 Interactive Charts – Visualize trends and patterns
	🎛️ Slicers & Filters – Enable dynamic data exploration
	⏱️ Screen Time Analysis – Monitor daily screen usage
	😴 Sleep Quality Analysis – Compare sleep patterns
	💪 Exercise Analysis – Evaluate physical activity trends
	🌍 Country-wise Insights – Analyze data by country
	👥 Gender-wise Analysis – Compare behavior across genders
	🎓💼 Student vs. Working Comparison – Explore lifestyle differences
	💡 Actionable Insights – Support data-driven decision-making.

 

📈 Key Insights :
⏱️ Higher screen time is associated with poorer sleep quality.
😴 Better sleep quality is linked to improved mental wellness.
💪 Regular exercise supports lower stress and better well-being.
📱 Excessive digital usage may increase stress levels.
👥 Digital lifestyle patterns vary across different user groups.

💼 Business Value :
This project provides data-driven insights that help organizations understand the impact of digital lifestyle habits on Gen Z mental wellness. The findings support the development of targeted wellness programs, encourage healthier technology usage, and enable personalized recommendations to improve overall well-being.

🚀 Future Enhancements :
